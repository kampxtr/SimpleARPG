# NotSimpleARPG
100 lines code make big 3D ARPG

<br>


# engine
- https://github.com/kampxtr/LiberatorEngine
## new features
- new animation system
- animatin blueprint
- lua script supported, all game code can be dumped to lua component


# gameplay
- battle system
- loot 
- inventory system
- equipment system
- multi-level
